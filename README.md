# 😊 CustomerPulse AI

AI-powered customer feedback analytics dashboard built with Streamlit, Groq Llama 3.3, Plotly, and ReportLab.

## 🚀 Features

- 📊 Interactive Dashboard
- 😊 Sentiment Analysis
- 🔥 Top Complaints Detection
- 🤖 AI Review Analyzer
- 📋 AI Executive Report
- 📄 PDF Report Generation
- ⬇ Download Filtered CSV
- 💬 AI Business Assistant
- 🎯 Dynamic Filters

## 🛠 Tech Stack

- Python
- Streamlit
- Plotly
- Pandas
- Groq API (Llama 3.3)
- ReportLab

## Installation

```bash
git clone https://github.com/abhinavsahu2739-art/CustomerPulse-AI.git

cd CustomerPulse-AI

pip install -r requirements.txt

streamlit run dashboard/app.py
```

## Environment Variables

Create a `.env` file.

```env
GROQ_API_KEY=your_api_key_here
```
